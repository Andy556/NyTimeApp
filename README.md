# NyTimeApp
Download this complete Project in local system. 
Open Android Studio and click on Import Android Project, and give the resptive path in the import window.
Build the code before you run.
Now, Connect Android Data cable to your system and connect to Android phone.
Click on Run from Studio, you can find your device. If its not found then Goto Developer OPtion in Settings of Phone and then enable USB debugging
After that when you are device is visible on the studio.
Run the App.
